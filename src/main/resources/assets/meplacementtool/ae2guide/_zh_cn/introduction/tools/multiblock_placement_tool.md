# 多方块放置工具

<ItemImage id="meplacementtool:multiblock_placement_tool" scale="4" />

多方块放置工具是一款强大的批量放置工具，让你能够一次性放置多个相同的方块。它会自动沿着同类型方块表面扩展放置，非常适合快速建造大型结构。

## 功能特性

- **批量放置**：一次放置 1/8/64/256/1024 个方块
- **智能扩展**：自动沿着同类型方块表面扩展放置
- **网络集成**：通过 <ItemLink id="ae2:wireless_access_point" /> 连接到你的 ME 网络
- **18个配置槽位**：可配置最多18种不同物品，快速切换
- **撤销支持**：支持撤销最近的放置操作
- **内存卡支持**：副手手持 <ItemLink id="ae2:memory_card" /> 时，放置设备时自动应用其设置
- **Mekanism配置卡支持**：副手手持 Mekanism 配置卡时，放置设备时自动应用其设置

## 操作指南

- **手持工具右键（空气中）**：打开配置界面
- **手持工具右键（方块上）**：放置方块
- **按住 G**：打开轮盘菜单
- **轮盘内左键单击**：选择物品/数量

![轮盘菜单](../../assets/multiblock_placement_tool_radial.png)

- **手持工具 Ctrl + 左键**：撤销上一次放置操作
