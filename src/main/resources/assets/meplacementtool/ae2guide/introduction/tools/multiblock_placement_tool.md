# Multiblock Placement Tool

<ItemImage id="meplacementtool:multiblock_placement_tool" scale="4" />

The Multiblock Placement Tool is a powerful batch placement tool that allows you to place multiple identical blocks at once. It automatically expands along surfaces of the same block type, perfect for quickly building large structures.

## Features

- **Batch Placement**: Place 1/8/64/256/1024 blocks at once
- **Smart Expansion**: Automatically expands along surfaces of the same block type
- **Network Integration**: Connects to your ME network via <ItemLink id="ae2:wireless_access_point" />
- **18 Configuration Slots**: Configure up to 18 different items for quick switching
- **Undo Support**: Supports undoing recent placement operations
- **Memory Card Support**: When holding <ItemLink id="ae2:memory_card" /> in off-hand, automatically apply its settings when placing devices
- **Mekanism Configuration Card Support**: When holding Mekanism Configuration Card in off-hand, automatically apply its settings when placing devices

## Usage Guide

- **Right-click with tool (in air)**: Open configuration GUI
- **Right-click with tool (on block)**: Place blocks
- **Hold G**: Open radial menu
- **Left-click in radial menu**: Select item/quantity

![Radial Menu](../../assets/multiblock_placement_tool_radial.png)

- **Hold Ctrl + Left-click with tool**: Undo last placement operation
