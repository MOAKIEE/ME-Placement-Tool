# Changelog - Version 2.0.0-forge1.20.1

## English

### ✨ Added
1. **ME Cable Placement Tool** - A new tool for efficiently placing AE2 cables with three modes (line, fill, branch), smart dyeing system, and Key of Spectrum upgrade support.
2. **GUI Texture Overhaul** - Updated GUI textures.
3. **Placement Preview Colors** - Updated preview colors for block placement.
4. **Button Sound Effects** - Added audio feedback for GUI buttons.
5. **Tool HUD Display** - Shows item name and placement count when switching to tools, auto-hides after 2 seconds.
6. **In-Game Guide** - Added comprehensive guide pages for all tools.
7. **REI Compatibility** - Added support for Roughly Enough Items (REI) alongside existing JEI integration.

### ⚡ Improved
1. **Placement Sound System** - Placement sounds now correspond to the actual block being placed.
2. **Multiblock BFS Algorithm** - Optimized the BFS algorithm for the ME Multiblock Placement Tool, improving placement performance and accuracy.

# 更新日志 - 版本 2.0.0-forge1.20.1

### ✨ 新增
1. **ME 线缆放置工具** - 全新的线缆放置工具，支持直线、填充、分支三种模式，智能染色系统，以及"光谱的钥匙"升级支持。
2. **GUI 材质更新** - 更新了 GUI 材质。
3. **放置预览颜色优化** - 更新了方块放置预览颜色。
4. **按钮音效** - 新增 GUI 按钮交互音效。
5. **工具 HUD 显示** - 切换至工具时显示物品名称和放置数量，2 秒后自动隐藏。
6. **游戏内指南** - 添加了完整的工具使用指南。
7. **REI 兼容** - 新增对 Roughly Enough Items (REI) 的支持，与现有 JEI 集成并存。

### ⚡ 优化
1. **放置音效系统** - 放置音效现在会根据实际放置的方块类型播放对应的声音。
2. **多方块工具 BFS 算法** - 优化了 ME 多方块放置工具的 BFS 算法，提升了放置性能和准确性。
